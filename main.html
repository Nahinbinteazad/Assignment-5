Question-1: What are the different ways to select an element in the DOM?

There are a lot of ways

const d= document.getElementById('dec'); - Its by ID name
const d= document.getElementByClass('dec'); - Its by Class name
const d= document.getElementsByTagName('div'); - Its by Tag name
const d= document.querySelector('.dec'); - Its by Single name
const d= document.querySelectorAll('.dec'); - Its by All Query name

Question-2: What is the difference between innerHTML, innerText, and textContent ?
innerHTML : It returns the elements/contents inside of the html . All the p tag,li tag etc are included here .
innerHTML : It returns the text inside of the element . All the texts are included , its not going to work on Html tags.
textContent : Powerful than innerHTML . It can also work with hidden text but do not work on Html tags.
